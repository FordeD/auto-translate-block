# Auto Translate Block

Расширение для Chrome, которое автоматически переводит текст внутри выбранного HTML-блока на странице.

## Возможности

- **Выбор элемента**: Кликните для выбора любого HTML-элемента на странице для перевода
- **Визуальная подсветка**: Подсветка элемента под курсором во время выбора
- **Локальный прокси-сервер**: Node.js сервер с @iamtraction/google-translate (без CORS и капчи)
- **Умное разбиение текста**: Автоматическое разбиение длинных текстов на предложения
- **HMAC-аутентификация**: Защищённая связь между расширением и сервером
- **Настройки**: Языки перевода, цвет выделения, прозрачность
- **Перевод атрибутов**: Опциональный перевод alt, title и других атрибутов
- **Уведомления**: Визуальная обратная связь о статусе перевода
- **Локализация**: Интерфейс на английском и русском языках

## Установка

### 1. Установка прокси-сервера

```bash
cd ~/auto-translate-block/server/
npm install
```

### 2. Запуск прокси-сервера

```bash
npm start
```

Сервер запустится на `http://localhost:3000`

### 3. Установка расширения

1. Откройте Chrome и перейдите на `chrome://extensions/`
2. Включите "Режим разработчика" (переключатель в правом верхнем углу)
3. Нажмите "Загрузить распакованное"
4. Выберите папку `auto-translate-block/extension`

## Использование

1. **Запустите прокси-сервер** (должен работать до использования расширения)
2. **Выберите языки**: Нажмите на иконку расширения и выберите исходный и целевой языки
3. **Выберите элемент**: Нажмите кнопку "📍 Выбрать HTML элемент"
4. **Наведите курсор**: Проведите курсором по странице — элементы будут подсвечиваться
5. **Кликните**: Нажмите на элемент, который нужно перевести
6. **Готово**: Текст внутри выбранного элемента (и всех вложенных) будет переведён

### Горячие клавиши

- **ESC**: Отменить выбор элемента

## Настройки

Откройте настройки, нажав на значок ⚙️ в popup-окне расширения.

### Языковые настройки

- **Язык интерфейса**: English / Русский
- **Исходный язык**: Язык исходного текста (или "Автоопределение")
- **Целевой язык**: Язык, на который переводить

### Внешний вид

- **Цвет выделения**: Цвет подсветки при выборе элемента
- **Прозрачность выделения**: Уровень прозрачности подсветки

### Дополнительно

- **Сохранять HTML форматирование**: Сохранять форматирование текста
- **Переводить атрибуты alt и title**: Переводить атрибуты элементов
- **Показывать уведомления**: Показывать уведомления о переводе

## API прокси-сервера

Расширение использует локальный прокси-сервер для перевода:

```
Расширение → POST http://localhost:3000/translate → Google Translate
```

### POST /translate

**Тело запроса**:
```json
{
  "text": "Hello World",
  "from": "en",
  "to": "ru"
}
```

**Ответ**:
```json
{
  "success": true,
  "translatedText": "Привет Мир",
  "original": "Hello World"
}
```

### Безопасность

- **HMAC-SHA256 подпись**: Все запросы подписываются секретным ключом
- **Защита от replay-атак**: Уникальный nonce + timestamp
- **Секрет в background**: Секретный ключ хранится только в service worker

## Требования

- Chrome 88 или новее (для Manifest V3)
- Node.js 14+ (для прокси-сервера)
- Подключение к интернету для перевода
